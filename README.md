# Expense Tracker

Expense Tracker is a lightweight Rails app for keeping tabs on day-to-day spending and splitting costs with friends or family. Log expenses, organize them by group, and keep everyone on the same page without spreadsheets.

- Track personal and shared expenses in one place
- Invite others to collaborate on group budgets
- Get quick summaries that highlight where your money goes

Try it here: https://expense-tracker-tamu-7931d3da2d70.herokuapp.com/

To run it locally, install Ruby 3.3, run `bundle install`, and start the server with `bin/rails server`.
